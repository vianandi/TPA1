Documentation of Calculator App

This project was created to help users perform various arithmetic calculations. This android application will have a user interface with numbers and arithmetic operations.

There are several functions available in this application, including :

  • Numbers created using buttons (Button)
  • Operators +, -, /, *, % are made using buttons (Button)
  • Create buttons for delete (C, AC, Del) and answer ( = )
  • The screen will display the number entered by the user and the result will come out if the user clicks answer ( = )
  • Once users have installed it, they are always ready to use it
