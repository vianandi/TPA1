Documentation of E-Commerce App

This project is temporarily created to help people who have an offline store and want to sell their products online.

This application is run by 3 people, namely :

  • Admin
  • Staff
  • User

Some of the features available in this application are :

  • Admin :
    • Add staff
    • Add items
    • Manage databases
  • Staff :
    • Has login pages
  • User :
    • Can register & login
    • Create product views
