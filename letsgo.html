Documentation of Bus Ticket Booking App

This project was created to make it easier for users to travel by bus. The bus seat reservation system is concerned with maintaining the details of each passenger.

There are several features available in this application, including :

  • Animated Splash Screen
  • Animated Login
  • Animated Register
  • Home Page
  • Booking Page
  • Bus Details
  • History Booking
